To install and run this app folow steps below.

1. Install and setup mysql server (port 3306).
2. Create DB from "snip_db.sql"
3. In "Snuppy.jar/hibernate.cfg.xml" change "YOURPASSWORD" with your mysql root pasword.

4. Run "Snuppy.jar".
(If previous the steps were done successfully, it should appear control panel)
5. Click "Start Server".
(Now your server runs locally , and you can access app)
6. Open link "localhost:8080/reg".
7. Register & download app.
8. Unzip and run "Snuppy.exe" from "Snuppy.zip".
9. Login and enjoy app.
